Stock Valuation Workspace

티커를 입력하면 가치평가의 근거와 한계를 함께 보여줍니다

DCF 절대가치와 PER·PBR 상대가치를 한 화면에서 비교하되, 예측 변수는 적게 유지합니다. 이 웹페이지는 https://sonchanggi.github.io/valuation/에 배포되는 새로운 정적 프로젝트이며, 기존 프로젝트 결과물은 참고만 하고 수정하지 않습니다.

판단은 사용자에게 있습니다

이 화면은 투자 의견이 아니라, 사용자의 판단을 정리하기 위한 계산 보조 도구입니다. 입력값과 데이터 품질에 따라 결과는 크게 달라질 수 있습니다.

Static Data Contract

지원 티커는 생성된 JSON에 들어있는 회사입니다

GitHub Pages는 백엔드나 비밀 토큰 없이 동작합니다. 따라서 브라우저가 SEC나 가격 제공자에 직접 접속하지 않고, scripts/update_data.py와 GitHub Actions가 만든 같은 출처의 docs/data/*.json만 읽습니다. 새 티커가 필요하면 데이터 생성 스크립트나 수동 워크플로를 실행하세요.

Methodology & Limits

방법론과 데이터 한계

DCF 절대가치

최근 자유현금흐름을 기준으로 5년 명시 성장률, 할인율, 영구성장률만 주요 예측 변수로 둡니다.

PER · PBR 상대가치

EPS와 BPS에 사용자가 확인한 비교 배수를 적용합니다. 기본 배수는 예시값이며, P/S와 P/FCF는 보조 지표로만 사용합니다.

데이터 정책

SEC EDGAR 재무 데이터는 baseline, 시장가격은 best-effort입니다. 브라우저는 외부 금융 API를 직접 호출하지 않습니다.

본 페이지는 개인 리서치와 학습을 위한 정적 가치평가 도구입니다. 투자, 세무, 법률 또는 매매 조언이 아닙니다. 결과는 데이터 지연, 회계 태그 차이, 사용자의 가정에 따라 달라질 수 있습니다.
